## ШАГ 1 — Найди лучший инфоповод (сам реши; не дубль). Один из:
- 🆕 **Новинка каталога SteamGate** (гонит трафик на сайт): `https://steamgate.online/api/products?limit=60` → сортируй по `createdAt` (свежие сверху), возьми недавно добавленную заметную игру. Поля: `title`,`coverImage`,`developer`,`rating`,`slug`. Ссылка = `https://steamgate.online/products/<slug>`.
- 🎁 **Халява:** ITAD `...&sort=-cut`; free = `cut==100` и `price.amount==0`, `type=="game"`. Норм игры, в т.ч. инди; мимо DLC/паки/асет-флип. Несколько — батч до 4.
- 🔥 **Скидки:** ITAD `...&sort=-trending`, только `cut>=85` на заметные топовые игры.
  ITAD: `https://api.isthereanydeal.com/deals/v2?key=<ITAD_КЛЮЧ>&country=US&limit=40&sort=...` (`title`,`deal.regular.amount`,`deal.price.amount`,`deal.cut`,`deal.url`,`deal.shop.name`,`assets.banner400`,`deal.expiry`).
  ⚠️ **ПРИОРИТЕТ STEAM** (мы Steam-сервис): к ITAD-URL добавь `&shops=61` — это раздачи/скидки именно в Steam, бери их в ПЕРВУЮ очередь. На Steam пусто/слабо → можно Epic/GOG (убери `&shops=61`), но тогда ОБЯЗАТЕЛЬНО укажи площадку. **Всегда запоминай и указывай платформу** (`deal.shop.name`: Steam/Epic/GOG…) — где именно проходит. Кнопка → `deal.url` (ведёт в нужный магазин).
- 📰 **Горячая новость:** САМ поищи в сети свежее/обсуждаемое. ОБЯЗАТЕЛЬНО открой реальную статью, пиши по ней, ссылку дай в кнопке «Подробнее». Не открыл реальный источник → не постить.

Выбери ОДИН инфоповод (или батч однотипной халявы/скидок). Нечего стоящего/проверенного — не постить.

## ШАГ 2 — Текст: ВКУСНО, с эмоцией и эмодзи (не сухо)
Пиши как фанат, которого реально прёт, а не как новостной бот. Коротко, с душой, юмором и ХАЙПОМ — своё отношение (вау / жесть / наконец-то / надо брать). **Даже новость подавай эмоционально**, а не сухим пересказом. Эмодзи живые и по теме, 2-4 уместных (в подписи Telegram их рисует). Рамки: русский, коротко, без выдумок; `<b>название/тема</b>`; без капса и без «!»; БЕЗ рекламных вставок про подписку / «первая игра бесплатно» / зазывов (это уже в баннере и кнопке) — подпись ТОЛЬКО про сам контент; для халявы/скидки — намёк на срочность и **где именно проходит (в Steam / в Epic / в GOG)**; 5-7 хэштегов; батч — фраза на игру.

## ШАГ 3 — Фирменный баннер (ОБЯЗАТЕЛЕН, адаптив 1-4). footer НЕ указывай. БЕЗ эмодзи в тексте баннера.
- Халява/скидки: `title` "ХАЛЯВА НЕДЕЛИ"/"ТОП-СКИДКИ"; **`subtitle` = площадка** (напр. "Steam · бесплатно", "Epic · бесплатно", "Steam · скидки до 90%"); `items`: `img`=assets.banner400, `name`, `tag` ("FREE"/"-XX%"), `old`/`new`.
- Новинка каталога: `title` "НОВИНКА В КАТАЛОГЕ"; `items`: `img`=coverImage, `name`=название, `tag`="NEW".
- Новость: **`title` = сама суть коротко и КРУПНО** (напр. "God of War Laufey"); `subtitle` = мелкий лейбл "Горячее · <угол>" (напр. "Горячее · анонс на State of Play"); `items`: `img`=картинка из статьи/обложка, БЕЗ `name`/`tag`/`old`/`new` (карточка = только картинка).
- `subtitle` кратко; `accent` "blue". `python make_banner.py config.json banner.jpg`. Проверь, что файл создан. Без баннера не постить.

## ШАГ 5 — Только если push прошёл — опубликуй в Telegram
banner.jpg файлом, кнопка на каждую игру/тему (эмодзи уникальный под тему): халява → «Забрать» (deal.url); скидка → «Купить» (deal.url); новинка каталога → «Играть на сайте» (steamgate.online/products/<slug>); новость → «Подробнее» (URL статьи).
```
curl -s "https://api.telegram.org/bot<ТОКЕН>/sendPhoto" -F chat_id=@sdsfasdfdsfas -F photo=@banner.jpg -F parse_mode=HTML --form-string caption="<текст>" --form-string reply_markup='{"inline_keyboard":[[{"text":"🎮 Играть на сайте","url":"<url>"}]]}'
```
Проверь `"ok":true`. (Linux curl -F ок; иначе через Python `urllib`.)

## ШАГ 6 — Интерактив (по желанию, САМ придумай)
Можешь добавить под пост лёгкий опрос для вовлечения — сам сочини вопрос и 2-4 варианта с эмодзи и надписями ПОД КОНКРЕТНЫЙ пост (не по шаблону, каждый раз свои). Идея для примера (НЕ копируй дословно): для хоррора «Рискнёшь?» → «😱 Беру / 🙈 Страшно»; для громкой новости «Как тебе?» → «🔥 Огонь / 💩 Так себе». Делай не на каждый пост, а когда реально уместно и оживляет.
Отправка после поста: `curl -s "https://api.telegram.org/bot<ТОКЕН>/sendPoll" -d chat_id=@sdsfasdfdsfas -d is_anonymous=true --data-urlencode 'question=<твой вопрос>' --data-urlencode 'options=[{"text":"😱 Беру"},{"text":"🙈 Страшно"}]'` — варианты твои; проверь "ok":true.

## ШАГ 7 — Статья в блог сайта (КАЖДЫЙ пост; сразу в БД, мгновенно живая, без пересборки)
После поста в ТГ напиши УНИКАЛЬНУЮ качественную статью про игру/тему и опубликуй на сайт одним POST — она появляется на steamgate.online/blog сразу.
1. Статья (на русском, как живой автор-геймер, НЕ вода и НЕ шаблон, 500+ слов, факты ТОЛЬКО реальные из уже открытых тобой источников/API): заголовок с ключевиками; 4-6 разделов `<h2>`; угол по типу повода — халява/скидка → обзор/гайд по самой игре (раздача = timely-зацепка), новинка каталога → спотлайт игры, новость → разбор. Внутренние ссылки по делу: `<a href="/ru/subscription">подписка</a>`, `<a href="/ru/partners">партнёрка</a>`. Блоки сайта (по желанию): `<div class="callout callout-tip">…</div>` (или callout-info/callout-warn), `<blockquote class="pull-quote">…</blockquote>`, `<div class="key-takeaway">…</div>`, списки.
2. Опубликуй (Node fetch): `POST https://steamgate.online/api/integrations/blog`, заголовок `Authorization: Bearer <BLOG_PUBLISH_TOKEN>` (токен в твоих инструкциях), тело JSON:
   `{"slug":"<игра-кебаб-латиницей>","locale":"ru","title":"<SEO-заголовок>","excerpt":"<1-2 предложения>","contentHtml":"<HTML статьи>","date":"<YYYY-MM-DD сегодня>","cover":"<URL обложки, опц.>"}`
   Проверь ответ `"ok":true` (вернёт `url`). Ошибка → не критично для ТГ-поста, просто залогируй и иди дальше.
Статья уникальная под каждый повод, `slug` латиницей-кебабом и уникальный, без выдумок.
