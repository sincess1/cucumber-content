## ШАГ 0 — Дедуп по файлу `posted.json` (в корне этого репо)
- Узнай сегодняшнюю дату: `date +%Y-%m-%d`.
- Прочитай `posted.json`. Формат: `{"posted":[{"name":"Игра","date":"YYYY-MM-DD"}, ...]}`.
- Составь список игр, которые постились за последние **7 дней**. Эти игры повторно НЕ публикуй ни при каких условиях.

## ШАГ 1 — Собери инфоповод (приоритет халяве). Источник IsThereAnyDeal:
**А) Халява:** `GET https://api.isthereanydeal.com/deals/v2?key=<ITAD_КЛЮЧ>&country=US&limit=40&sort=-cut`
  Бесплатные = `cut == 100` и `price.amount == 0`. Бери ТОЛЬКО реальные заметные игры (`type == "game"`, магазин Steam/Epic/GOG). Пропускай DLC, паки, стартер-паки, курсы-бандлы (Fanatical bundle и подобный мусор).
**Б) Если новой халявы нет — скидки:** тот же URL с `sort=-trending`. Бери ТОЛЬКО `cut > 85` на заметные топовые игры. Скидки 85% и ниже — НЕ брать.

Убери из выборки всё, что в дедупе (шаг 0).
**Батч:** собери до 4 игр одного типа (халява ИЛИ большие скидки). Если их 3-4 — один пост на все. 1-2 — пост на 1-2. Больше 4 не брать. Если после дедупа ничего не осталось — НЕ постить.
Для каждой игры возьми: `title`; `deal.cut`; `deal.price.amount`; `deal.regular.amount`; ссылка = `deal.url`; обложка = `assets.banner400`; дедлайн = `deal.expiry`.

## ШАГ 2 — Напиши пост (русский, HTML, С ЭМОДЗИ)
- В заголовке эмодзи (🎁 для халявы / 🔥 для скидок).
- Каждая игра — отдельная строка: тематический эмодзи + `<b>Название</b>` + короткое описание одной фразой.
- Дедлайн: `⏳ <b>...</b>`. Для скидки цена: `💸 <s>старая</s> → <b>новая</b> (-XX%)`.
- Между блоками пустая строка.
- P.S.: `🥒 у нас 2000+ игр Steam по подписке, первая бесплатно — каталог в закрепе`.
- 5-7 хэштегов.

Пример (1 игра):
```
🎁 <b>Tell Me Why</b> бесплатно в Steam — нарративный детектив про близнецов.

Аляскинский городок, сверхъестественные способности, семейные тайны. Ни одного выстрела — просто история, что затягивает сильнее, чем ждёшь.

⏳ <s>$19.99</s> → <b>бесплатно до 1 июля</b>.

🥒 P.S. у нас 2000+ игр Steam по подписке, первая бесплатно — каталог в закрепе.

#халява #steam #бесплатно #tellmewhy
```

## ШАГ 3 — Фирменный баннер (до 4 карточек)
`make_banner.py` уже лежит рядом в этом репо.
- `pip install pillow -q` (если pillow не стоит).
- Создай `config.json` (для халявы `title` = "ХАЛЯВА НЕДЕЛИ", `tag` = "FREE"; для скидок `title` = "ТОП-СКИДКИ", `tag` = "-XX%"):
```
{"title":"...","subtitle":"<магазины · бесплатно или скидки>","accent":"blue","footer":"Первая игра бесплатно  ·  @cucumber_game","items":[{"img":"<assets.banner400>","name":"<игра>","tag":"FREE"}]}
```
- `python make_banner.py config.json banner.jpg`
- Открой `banner.jpg` и ПОСМОТРИ глазами: читаемо, обложки на месте, не криво. Плохо — поправь config и перерисуй.

## ШАГ 4 — Опубликуй `banner.jpg` файлом, кнопка на каждую игру
```
curl -s "https://api.telegram.org/bot<ТОКЕН>/sendPhoto" -F chat_id=@sdsfasdfdsfas -F photo=@banner.jpg -F parse_mode=HTML --form-string caption="<твой текст>" --form-string reply_markup='{"inline_keyboard":[[{"text":"🎮 Забрать <игра1>","url":"<url1>"}],[{"text":"🎮 Забрать <игра2>","url":"<url2>"}]]}'
```
По одной кнопке-строке на каждую игру. Проверь, что в ответе `"ok":true`. Ошибка — поправь и повтори.
