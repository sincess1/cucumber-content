## ШАГ 0 — Дедуп по `posted.json` (в корне репо)
- `date +%Y-%m-%d`. Прочитай `posted.json` (`{"posted":[{"name","date"}]}`). Игры за последние **7 дней** не публикуй повторно.

## ШАГ 1 — Собери инфоповод (приоритет халяве). IsThereAnyDeal:
**А) Халява:** `https://api.isthereanydeal.com/deals/v2?key=<ITAD_КЛЮЧ>&country=US&limit=40&sort=-cut` — бесплатные = `cut==100` и `price.amount==0`, только реальные заметные ИГРЫ (`type=="game"`, Steam/Epic/GOG); пропускай DLC, паки, стартер-паки, бандлы, шлак.
**Б) Если новой халявы нет — скидки:** тот же URL с `sort=-trending`, только `cut>=85` на заметные топовые игры.
Убери дубли и шлак. **Батч:** до 4 игр одного типа в один пост. После фильтра пусто — не постить.
Для каждой: `title`; `deal.regular.amount` (старая цена); `deal.price.amount` (новая); `deal.cut`; ссылка=`deal.url`; обложка=`assets.banner400`; дедлайн=`deal.expiry`.

## ШАГ 2 — Текст поста (русский, HTML, ЖИВО и ОБЪЁМНО)
Пиши как маркетолог, который реально любит игры: вкусно, с характером, чтобы захотелось забрать. Не сухо, но только правда.
- Заголовок-крючок с эмодзи (🎁 халява / 🔥 скидки) + название в `<b>...</b>`.
- **Описание** — 2-3 живых предложения: что за игра, какой вайб и ощущение от неё, чем цепляет, почему стоит хватать. С эмодзи.
- Короткая строка-ургентность: `⏳ успей до DD` (халява) или крючок по цене (скидка).
- **БЕЗ P.S. про подписку** (она зашита в баннер).
- 5-7 хэштегов.
- **Батч (2-4 игры):** на каждую — по 1-2 бойких предложения (короче, без стены текста), всё равно вкусно. Каждая игра — со своим эмодзи и `<b>названием</b>`.

Пример (1 игра, объёмно):
```
🎁 <b>Gravity Circuit</b> — забираем бесплатно в Steam 🔥

Представь Mega Man, только злее и быстрее: ты боевой робот Кай — носишься по неоновым уровням, цепляешься крюком-кошкой, вышибаешь боссов комбо и взламываешь врагов прямо в бою.

Ручной пиксель-арт, саундтрек, что застрянет в голове, и та самая затяжка «ну ещё один заход». Более 2 800 отзывов в Steam — почти все в восторге.

⏳ Успей забрать до 14 июня — потом снова станет платной.

#халява #steam #бесплатно #экшен #платформер #megaman #gravitycircuit
```
Пример строки в батче: `🎮 <b>Dead Cells</b> — роглайк-метроидвания, где смерть лишь начало: бешеный темп, сотни прохождений и оружие на любой вкус.`

## ШАГ 3 — Фирменный баннер (ОБЯЗАТЕЛЕН; адаптив 1-4: 1 большая / 2 в ряд / 3 = 2+1 / 4 = 2×2)
`make_banner.py` лежит рядом. `pip install pillow -q` если нет.
`config.json` (**footer НЕ указывай** — он автоматический: «Сыграй в любую новинку бесплатно · steamgate.online»):
- `title`: "ХАЛЯВА НЕДЕЛИ" / "ТОП-СКИДКИ".
- `subtitle`: платформа кратко ("Steam · бесплатно" / "Steam · скидки до 60%").
- `accent`: "blue".
- `items` по карточке: `img`=assets.banner400, `name`, `tag` ("FREE"/"-XX%"), `old`=старая цена ("$18.68"), `new`=новая ("бесплатно до 14 июня" / "$4.99").
```
{"title":"ХАЛЯВА НЕДЕЛИ","subtitle":"Steam · бесплатно","accent":"blue","items":[{"img":"<assets.banner400>","name":"Gravity Circuit","tag":"FREE","old":"$18.68","new":"бесплатно до 14 июня"}]}
```
`python make_banner.py config.json banner.jpg`. Открой banner.jpg, ПОСМОТРИ глазами (читаемо, обложки на месте, цена видна). Криво — поправь config. Без баннера не постить.

## ШАГ 4 — Опубликуй banner.jpg файлом, кнопка на каждую игру
```
curl -s "https://api.telegram.org/bot<ТОКЕН>/sendPhoto" -F chat_id=@sdsfasdfdsfas -F photo=@banner.jpg -F parse_mode=HTML --form-string caption="<текст>" --form-string reply_markup='{"inline_keyboard":[[{"text":"🎮 Забрать <игра1>","url":"<url1>"}],[{"text":"🎮 Забрать <игра2>","url":"<url2>"}]]}'
```
По кнопке-строке на игру. Проверь `"ok":true`. (Linux curl -F ок; если среда не даёт — отправь через Python `urllib`.)
