## ШАГ 0 — Дедуп по файлу `posted.json` (в корне этого репо)
- Узнай сегодняшнюю дату: `date +%Y-%m-%d`.
- Прочитай `posted.json` (`{"posted":[{"name","date"}]}`). Игры за последние **7 дней** повторно НЕ публикуй.

## ШАГ 1 — Собери инфоповод (приоритет халяве). Источник IsThereAnyDeal:
**А) Халява:** `GET https://api.isthereanydeal.com/deals/v2?key=<ITAD_КЛЮЧ>&country=US&limit=40&sort=-cut`
  Бесплатные = `cut == 100` и `price.amount == 0`. Только реальные заметные ИГРЫ (`type == "game"`, Steam/Epic/GOG). Пропускай DLC, паки, стартер-паки, курсы-бандлы, обскур-шлак.
**Б) Если новой халявы нет — скидки:** тот же URL с `sort=-trending`. Только `cut > 85` на заметные топовые игры. 85% и ниже — НЕ брать.

Убери всё из дедупа и всё неизвестное/шлак. **Батч:** до 4 игр одного типа в один пост (3-4 → один пост; 1-2 → пост на 1-2; больше 4 не брать). После фильтра пусто — НЕ постить.
Для каждой игры: `title`; `deal.regular.amount` (старая цена); `deal.price.amount` (новая); `deal.cut`; ссылка = `deal.url`; обложка = `assets.banner400`; дедлайн = `deal.expiry`.

## ШАГ 2 — Текст поста (русский, HTML, С ЭМОДЗИ)
Цена и дедлайн теперь рисуются НА БАННЕРЕ — в тексте их подробно НЕ дублируй. Текст короткий и живой:
- Заголовок с эмодзи (🎁 халява / 🔥 скидки).
- Каждая игра — строка: тематический эмодзи + `<b>Название</b>` + точное описание 1-2 фразами (реальный жанр/суть, без выдумок).
- **НЕ добавляй P.S. про подписку** — она уже зашита в баннер.
- 5-7 хэштегов.

Пример (1 игра):
```
🎁 <b>Gravity Circuit</b> — бесплатно в Steam.

Мега-мен в аниме-обёртке: пиксельный экшен-платформер, где ты боевой робот с цепным крюком. Быстро, жёстко, с боссами.

#халява #steam #бесплатно #экшен #платформер #gravitycircuit
```

## ШАГ 3 — Фирменный баннер (ОБЯЗАТЕЛЕН, до 4 карточек)
`make_banner.py` лежит рядом в репо. `pip install pillow -q` если нет.
Создай `config.json` (НЕ указывай footer — он автоматический «Первая игра бесплатно · steamgate.online»):
- `title`: "ХАЛЯВА НЕДЕЛИ" для халявы / "ТОП-СКИДКИ" для скидок.
- `subtitle`: платформа кратко, напр. "Steam · бесплатно" или "Steam · скидки".
- `accent`: "blue".
- `items` — по карточке на игру: `img` = assets.banner400, `name` = название, `tag` = "FREE" или "-XX%", `old` = старая цена строкой (напр. "$18.68"), `new` = новая (напр. "бесплатно до 14 июня" для халявы / "$4.99" для скидки).
```
{"title":"ХАЛЯВА НЕДЕЛИ","subtitle":"Steam · бесплатно","accent":"blue","items":[{"img":"<assets.banner400>","name":"Gravity Circuit","tag":"FREE","old":"$18.68","new":"бесплатно до 14 июня"}]}
```
Запусти РОВНО: `python make_banner.py config.json banner.jpg`. Открой `banner.jpg`, ПОСМОТРИ глазами (читаемо, обложки на месте, цена видна). Плохо — поправь config и перерисуй. Не сделался баннер — почини (pillow, python), без него НЕ постить.

## ШАГ 4 — Опубликуй `banner.jpg` файлом, кнопка на каждую игру
```
curl -s "https://api.telegram.org/bot<ТОКЕН>/sendPhoto" -F chat_id=@sdsfasdfdsfas -F photo=@banner.jpg -F parse_mode=HTML --form-string caption="<твой текст>" --form-string reply_markup='{"inline_keyboard":[[{"text":"🎮 Забрать <игра1>","url":"<url1>"}],[{"text":"🎮 Забрать <игра2>","url":"<url2>"}]]}'
```
По одной кнопке-строке на каждую игру. Проверь `"ok":true`. (На Linux curl -F работает; если среда не даёт — отправь тем же методом через Python `urllib`/`requests`.)
